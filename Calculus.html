What are composite functions

A function is composite if you can write it as:

f(g(x))f\big(g(x)\big)f(g(x))

In other words, it is a function within a function, or a function of a function.

For example, cos⁡(x2)\cos(x^2)cos(x2) is composite because if we let f(x)=cos⁡(x)f(x)=\cos(x)f(x)=cos(x) and g(x)=x2g(x)=x^2g(x)=x2, then cos⁡(x2)=f(g(x))\cos(x^2)=f(g(x))cos(x2)=f(g(x)).

g is the function within f, so we call g the “inner” function and f
the “outer” function.
